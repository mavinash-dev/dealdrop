## S21 — Mobile Polish & PWA (~2h)
**Checklist:**
- [ ] All pages usable at 375px (iPhone SE) — test every page
- [ ] Bottom nav doesn't overlap content (add `pb-20` to page content on mobile)
- [ ] No horizontal scroll anywhere
- [ ] Touch targets min 44px height

**PWA setup:**
```bash
npm install -D vite-plugin-pwa
```

**vite.config.js:**
```js
import { VitePWA } from 'vite-plugin-pwa'
export default {
  plugins: [
    react(),
    VitePWA({
      registerType: 'autoUpdate',
      manifest: {
        name: 'DealDrop',
        short_name: 'DealDrop',
        theme_color: '#4f46e5',
        icons: [{ src: '/icon-192.png', sizes: '192x192', type: 'image/png' }]
      }
    })
  ]
}
```

**Create:** `public/icon-192.png` (simple branded icon, any tool)

**Done when:**
- [ ] No layout breaks at 375px
- [ ] PWA installable on Android Chrome ("Add to Home Screen")
- [ ] `manifest.json` generated by Vite

## S23 — Seed 100+ Real Deals (~2h)
**Run agents manually first:**
```bash
curl -H "x-cron-secret: $CRON_SECRET" https://dealdrop.in/api/cron/sync-earnkaro
curl -H "x-cron-secret: $CRON_SECRET" https://dealdrop.in/api/cron/sync-reddit
```

**Then manually add 20–30 high-quality deals via Supabase dashboard** to fill gaps.

**Must cover (at least 5 deals each):**
- Food: Swiggy, Zomato, Blinkit
- Shopping: Amazon, Flipkart, Myntra
- Travel: MakeMyTrip, Cleartrip, IRCTC
- Payments: PhonePe, GPay, Paytm

**Done when:**
- [ ] 100+ rows in `community_deals` with `status = 'active'`
- [ ] At least 5 deals per category above
- [ ] All deals have an expiry date set

## S24 — Vercel Deploy & Domain (~1h)
**Steps:**
1. `git push` to `github.com/mavinash-dev/dealdrop`
2. Vercel Dashboard → New Project → Import from GitHub
3. Add all env vars from `.env.example` in Vercel settings
4. Vercel auto-deploys on every push to `main`
5. Vercel Dashboard → Domains → add `dealdrop.in`
6. Cloudflare DNS → add Vercel's CNAME record
7. Enable Vercel Analytics (one toggle in dashboard)

**Verify after deploy:**
- [ ] `https://dealdrop.in` loads app
- [ ] Google Sign-In works (update OAuth redirect URI to production URL)
- [ ] Cron jobs visible in Vercel → Cron tab
- [ ] Vercel Analytics recording visits
